Практичне завдання: UI Controls

Опис і приклади елементів інтерфейсу

1. RadioButton

Елемент, що дозволяє вибрати тільки один варіант із групи.

2. Checkbox

Елемент для вибору кількох варіантів одночасно.

3. Text Input

Поле для введення тексту користувачем.

4. Tabs

Вкладки дозволяють організувати контент у різних секціях.

5. Button

Кнопка виконує дію при натисканні.

6. Text Label

Текстовий напис, який пояснює інші елементи.

7. Link

Гіперпосилання для переходу на іншу сторінку чи ресурс.

9. Dropdown List

Випадаючий список для вибору одного значення.
